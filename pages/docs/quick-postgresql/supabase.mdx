# Supabase
According to their **[official website](https://supabase.com/docs)**, Supabase is an open source Firebase alternative for building secure and performant Postgres backends with minimal configuration.

Checkout the official **[Supabase + Drizzle](https://supabase.com/docs/guides/database/connecting-to-postgres#connecting-with-drizzle)** docs.

### Create your models

```typescript copy filename="schema.ts"
import { pgTable, serial, text, varchar } from "drizzle-orm/pg-core";

export const users = pgTable('users', {
  id: serial('id').primaryKey(),
  fullName: text('full_name'),
  phone: varchar('phone', { length: 256 }),
});
```

### Make your first query

```typescript copy filename="index.ts"
import { drizzle } from 'drizzle-orm/postgres-js'
import postgres from 'postgres'
import { users } from './schema'

const connectionString = process.env.DATABASE_URL
const client = postgres(connectionString)
const db = drizzle(client);

const allUsers = await db.select().from(users);
```

</Steps>

Connect to your database using the Connection Pooler for **serverless environments**, or by using the Direct Connection for **long-running servers**.

### Usage with Cloudflare Workers
