To toggle a column value you can use `update().set()` method like below:

<Section>
```tsx copy {8}
import { eq, not } from 'drizzle-orm';

const db = drizzle(...);

await db
  .update(table)
  .set({
    isActive: not(table.isActive),
  })
  .where(eq(table.id, 1));
```

```sql
update [table] set [is_active] = not [table].[is_active] where [table].[id] = 1;
```
</Section>

Please note that there is no boolean type in MSSQL, it uses the `bit` data type instead (`0` for false and `1` for true).
