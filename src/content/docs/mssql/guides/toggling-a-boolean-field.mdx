To toggle a column value you can use `update().set()` method with the `sql` operator like below:

<Section>
```tsx copy {8}
import { eq, sql } from 'drizzle-orm';

const db = drizzle(...);

await db
  .update(table)
  .set({
    isActive: sql`1 - ${table.isActive}`,
  })
  .where(eq(table.id, 1));
```

```sql
update [table] set [is_active] = 1 - [table].[is_active] where [table].[id] = 1;
```
</Section>

Please note that there is no boolean type in MSSQL, it uses the `bit` data type instead (`0` for false and `1` for true).
