Create a `index.ts` file in the `src` directory and initialize the connection:

<CodeTabs items={["node:sqlite", "node:sqlite with config"]}>
```typescript copy
import 'dotenv/config';
import { drizzle } from 'drizzle-orm/node-sqlite';

const db = drizzle(process.env.DB_FILE_NAME!);
```
```typescript copy
import 'dotenv/config';
import { drizzle } from 'drizzle-orm/node-sqlite';

// You can specify any property from the node:sqlite connection options
const db = drizzle({ connection: { source: process.env.DB_FILE_NAME! }});
```
</CodeTabs>

If you need to provide your existing driver:
```typescript copy
import 'dotenv/config';
import { drizzle } from 'drizzle-orm/node-sqlite';
import { DatabaseSync } from 'node:sqlite';

const sqlite = new DatabaseSync(process.env.DB_FILE_NAME!);
const db = drizzle({ client: sqlite });
```
